Figma Clipboard String Bridge

把你在 Figma 里复制的设计稿 粘贴进这个页面，页面会提取剪贴板中的 text/html 原始字符串。之后把这段字符串放回页面，并通过“复制为 Figma 内容” 或手动 Ctrl/Cmd + C，页面会把它重新写成带 text/html 的剪贴板内容。

1. 从 Figma 提取

  1. 在 Figma 中选中图层并复制。
  2. 点击下面的“读取系统剪贴板”，或者直接在粘贴区按 Ctrl/Cmd + V。
  3. 页面会显示 Figma 原始的 text/html 字符串，格式会更接近 clipboard-inspector。
把焦点放在这里后粘贴 支持直接粘贴。页面会捕获 Figma 的 text/html 内容并编码。

2. 还原回 Figma

说明：直接把文本框里的字符串当普通文本复制到 Figma，通常不会恢复成设计稿。要让 Figma 识别， 剪贴板里必须带有 text/html 类型。本页会在“复制为 Figma 内容”按钮或你手动复制时， 尝试把这段字符串重新写成富剪贴板数据。

说明：这个工具会优先保留 Figma 剪贴板中的 text/html 与 text/plain。 如果你是以 file:// 形式打开页面，部分浏览器的 Clipboard API 可能受限；此时优先使用手动复制， 让页面在 copy 事件里写入 HTML。
