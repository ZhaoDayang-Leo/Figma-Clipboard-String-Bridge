# Figma Clipboard String Bridge

A lightweight HTML tool for extracting Figma clipboard `text/html`, turning it into a shareable string, and restoring it back to clipboard content that Figma can paste.

## Live use

[Open `index.html` in a browser, or publish this repository with GitHub Pages.
](https://zhaodayang-leo.github.io/Figma-Clipboard-String-Bridge/)
